Availability & Payments
Live Chat availability

Control the days/hours Live Chat shows as online. Outside these hours (or when disabled below), visitors see the offline message and leave-a-message form instead.

Booking availability

Controls the "Book a call" form on /book.html — no external calendar account needed.

Payments (Paystack)

Get your keys from the Paystack dashboard (Settings → API Keys & Webhooks). Use test keys first — they start with pk_test_ / sk_test_.

Only the Starter tier gets a direct "pay now" checkout button — Growth and Custom stay quote-only since their scope varies. Leave the amount empty to hide the checkout button entirely.

Site Behavior
Site widgets

Show or hide the floating buttons on every public page — no need to remove the underlying contact details just to hide them temporarily.

Maintenance mode

When enabled, visitors see a "down for maintenance" page (503) on every public page. Admin, API, and uploads stay reachable so you can turn it back off.
